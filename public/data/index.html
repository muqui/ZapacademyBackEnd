API Sistema de Software registro de asistencias

Todas las urls a excepción de registro, login y crear usuario, necesitan autenticación:





Crear usuario: POST

https://zapacademy.herokuapp.com/register

{

"name":"",

"lastname": "",

"email":"",

"password": "",

"role": "Admin or User"

}



Login: POST

https://zapacademy.herokuapp.com/login

{

"email":"",

"password": ""

}

Lista de beneficairios por evento: GET



https://zapacademy.herokuapp.com/evento/eventos/{evento}



Lista de eventos por beneficiario: GET

https://zapacademy.herokuapp.com/evento/beneficiarios/{evento}



Lista de todos los eventos vigentes: GET



https://zapacademy.herokuapp.com/evento/eventos/



Regresa beneficiario a partir de su CURP y id del evento : GET

https://zapacademy.herokuapp.com/beneficiario/{:curp}/{:evento}





Valida si un beneficiario registró asistencia :POST

localhost:5000/evento/asistencia/verificar

Posibles resultados:

"no_exists"

"exists"



BODY

{

"event_id": 1,

"beneficiary_id":80

}
